所有命令需在项目根目录下执行：

| 命令                        | 作用                    |
| :-------------------------- | :---------------------- |
| `npm install`             | 安装依赖                |
| `npm run dev`             | 启动本地开发服务器      |
| `npm run build`           | 构建生产环境代码        |
| `npm run preview`         | 预览构建结果            |
| `npm run astro ...`       | 运行 Astro CLI 命令     |
|
